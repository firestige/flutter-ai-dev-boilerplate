# 项目文档索引 (Store Front for AI)

这份文档是 AI 助手了解项目的入口。请按照以下指引加载上下文。

## 📂 文档结构

### 1. 必读上下文 (Critical Context)
在开始任何任务前，请确保你了解以下文档的内容：
- **[README.md](../README.md)**: 项目背景、核心价值和目标。
- **[guideline.md](guideline.md)**: 开发规范、代码风格、Git 提交约定、AI 协作流程。
- **[todo.md](todo.md)**: 当前的任务规划和进度。

### 2. 按需加载 (On-Demand Loading)
根据任务的具体内容，查阅相关文档：

#### 🏗 架构与设计 (Design)
位于 `design/` 目录。
- **[design/architecture.md](design/architecture.md)**: **架构基本法**。包含核心分层原则、目录结构、通信机制定义。任何架构调整必须遵循此文档。
- **[design/index.md](design/index.md)**: 具体模块的设计概览、技术选型决策。
- 当涉及到新增功能、修改现有逻辑或重构时，**必须**查阅并更新设计文档。

#### 📜 变更与任务 (Changelog & Tasks)
- **[changelog/history.md](changelog/history.md)**: 记录 main 分支的版本变更历史。
- **[task/README.md](task/README.md)**: 归档已完成的复杂任务执行记录（Task Journal）。
- 了解项目演进路线和具体任务细节时查阅。

## 🚀 快速开始指引
1. 阅读 `README.md` 了解我在做什么。
2. 阅读 `docs/guideline.md` 了解你应该怎么做。
3. 查看 `docs/todo.md` 领取或确认当前任务。
4. 如果任务涉及设计变更，查阅 `docs/design/`。
